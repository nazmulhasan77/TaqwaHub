# Privacy Policy for TaqwaHub

Last updated: July 23, 2026

TaqwaHub stores settings and tracker data locally in Chrome storage. This includes location preferences, custom prayer times, alerts, Salah history, fasting history, Quran progress, Dhikr counts, tasks, notes, quick links, custom Hijri events, dashboard order and appearance settings.

When automatic location is enabled, the browser supplies approximate latitude and longitude after the user grants permission. Those coordinates are sent directly to the Aladhan prayer-time API to calculate prayer and Hijri data. TaqwaHub does not operate its own server and does not sell user data.

Quran, translation or other enabled content may be requested from the external API domains listed in the extension manifest. Backup export creates a JSON file on the user's device. Import reads only the file selected by the user.

Users can erase locally stored extension data with **Reset all data**. Uninstalling the extension also removes its Chrome-local data according to browser behavior.

Islamic dates may vary according to local moon sighting and official authorities.
